Infinity is a non-profit organization that manages Infinity cryptomoney. The foundation aims to guarantee money by holding assets and increasing its value.
The blockchain executes smart contracts: applications that work exactly as programmed, without any censorship, fraud or third-party interference.

Infinity software specializes in software development that uses the value of Infinity cryptomoney. Developing tools are the tools of trading and tools giving liquidity to foreign exchange markets. Wallet installation for Linux

Before we begin please download the wallet installation package from the official repository https://github.com/Plusid/infinity.

Clone locally

                                        cd infinity
                                         
                                        ls infinity

You will need the following dependencies:

And these optional development dependencies:

Library Purpose Description

Run these commands

What is DPoS

Delegated Proof-of-Stake (DPoS) is a more efficient PoS mechanism. DPoS uses a reputation system and real-time voting to achieve consensus.

Think of Delegated Proof of Stake as technological democracy. Community members vote for delegates to secure their network and delegates will be rewarded by validating transactions for the next block. Since the vote in DPoS is always on-going, if the representative does not behave or perform well, people in the community can remove their votes, essentially firing the bad representative.

The representative must always try their best to obey all rules and keep the network uptime as high as possible otherwise the representative may be overthrown by its community members.

To be a delegate, a INF holder needs to pick a name that have not be used before and register as a delegate. The process can be accomplished in either the full node or light wallet, but keep in mind that the forging process is only possible in the full node wallet. This means that you can register a delegate in either version of the wallet but will only be able to perform the delegate functions in a full version of the client. All INF accounts are eligible to become delegates.

What is Peer-to-Peer

We are using a standard infinity P2P network and Protocol https://en.wikipedia.org/wiki/Peer-to-peer. All peer-to-peer communication occurs entirely over TCP.

In a Peer to Peer network, the "peers" are computer systems which are connected to each other via the Internet. Files can be shared directly between systems on the network without the need of a central server. In other words, each computer on a P2P network becomes a file server as well as a client.

What are Network Fees

All valid transactions in the network must be processed. Delegates process transactions and store them in new blocks and receive fees as rewards. For this work, the delegates receive a fee. All transactions in the network must contain some type of fee as a spam countermeasure.

The minimum network fee for sending an INF transaction is 0.00001 INF. The amount depends on the size of this transaction. For example, a 1 INF transaction includes an additional fee of 0.00001 INF, so it cost 1.00001 INF totally.

The following is a list of fees for different types of transactions:

  • 0.00001 INF of amount sent for a spend transaction
  • 1 INF for registering delegate name
  • 0.01 INF for voting delegates
  • 0.01 INF for cancelvoting delegates

Delegates receive fees from all transactions of the block. Delegates who missed creating a block assigned to them during that cycle are not paid.

What is a full node wallet?

Full nodes are nodes that store the entire INF blockchain and are connected in a P2P network architecture. In the blockchain network, all full nodes are treated as equals, and serve as both clients and servers.

The infinity full node consists of two programs:
• infinityd: The core wallet program;
• infinity-cli: a command line interface;
